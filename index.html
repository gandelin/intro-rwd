Introduction to Responsive Web Design (RWD)

What is Responsive Web Design?

Responsive Web Design means to design your website in such a way that it is responsive to all devices accessing the web.

The Problem?

Currently there are so many device types and sizes accessing the internet. (There are currently over 24,093 distinct Android devices available on the market alone.) To make your website viable you need to be responsive to all different screen sizes. If the user experience is poor on your website, people won't use it causing you to lose business.

The Solution

  • A flexible, grid-based layout

    The website needs to be designed with flexibility in mind. The grid-based archetecture is ideal for this. For small mobile devices your website can provide the basic functionality. For larger screen sizes the user experience can be enhanced.

  • Flexible Images and Media

    The images and media provided also need to be responsive the various screen sizes and internet speeds.

  • Media Queries

    A module from the CSS3 specification
